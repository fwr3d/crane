# Crane v1.1 — Full Feature List

## Priority 1: Core UX (Week 1)

### 1. Mobile responsive layout
- Breakpoints for tablet/phone
- Stack columns vertically on mobile
- Touch-friendly drag/drop
- Hamburger menu for navigation

### 2. Board filtering & sorting
- Filter by: status, company name, date range
- Sort by: deadline, date added, company A-Z
- Search bar across all jobs
- "Clear filters" button

### 3. Keyboard shortcuts
- A - Add new job
- F - Focus search
- Esc - Close modals
- Arrow keys - Navigate cards
- E - Edit selected card

### 4. Activity log
- Track: status changes, note edits, deadline updates
- Show in job detail panel
- Timestamp + action type
- Export activity with CSV


## Priority 2: Job Discovery (Week 2)

### 5. Multi-source job aggregation
- Greenhouse RSS scraper
- Lever job boards
- RemoteOK API integration
- Y Combinator Work at a Startup
- Manual URL paste (with metadata extraction)

### 6. Better LinkedIn integration
- Save full LinkedIn job URL
- Extract company/position from URL
- One-click "Apply on LinkedIn" button
- Track application source

### 7. Job tagging system
- Create custom tags (Remote, Frontend, Startup, etc.)
- Filter by tags
- Color-coded tag badges
- Bulk tag multiple jobs


## Priority 3: Notifications & Automation (Week 3)

### 8. Email reminders
- Daily digest: stale applications (14+ days)
- Weekly summary: stats + action items
- Deadline alerts (3 days before)
- Opt-in/out per notification type

### 9. Smart follow-up tracking
- Auto-calculate "follow up by" dates
- Status-based reminder rules (Applied → follow up in 7 days)
- Snooze reminders
- Mark as "waiting for response"

### 10. Calendar integration
- Export interview dates to .ics
- Google Calendar sync (optional)
- Deadline reminders in calendar


## Priority 4: Richer Job Details (Week 4)

### 11. Enhanced job data
- Salary range field
- Location (Remote/Hybrid/Onsite)
- Job type (Full-time/Contract/Internship)
- Referral contact info
- Application materials used (which resume version)

### 12. Interview tracking
- Multiple interview rounds
- Round type (Phone screen, Technical, Onsite, etc.)
- Interviewer names
- Interview notes per round
- Outcome per round

### 13. Document attachments
- Upload resume/cover letter per job
- Link to tailored resume versions
- Store job descriptions as PDF
- Version control for application materials


## Priority 5: Polish & Analytics (Week 5)

### 14. Advanced stats dashboard
- Response rate by company size
- Time-to-response metrics
- Success rate by job source
- Application velocity graph (jobs/week)
- Offer conversion funnel

### 15. Dark mode
- Toggle in settings
- Persist preference
- Smooth transitions
- All components themed

### 16. Collaboration features (optional)
- Share board with mentor/career coach (view-only)
- Export shareable link
- Anonymous mode (hide personal info)

### 17. Data management
- Bulk actions: delete, archive, export
- Archive old applications (keep stats, hide from board)
- Import from CSV/LinkedIn exports
- Backup/restore database


## Priority 6: Performance & Reliability

### 18. Better database
- Migrate SQLite → PostgreSQL (Railway supports it)
- Prevents data loss on redeploys
- Better concurrent access
- Scheduled backups

### 19. Caching & optimization
- Cache company logos
- Lazy load job cards
- Infinite scroll on Find page
- Debounce search inputs

### 20. Error handling
- Retry failed job scrapes
- Offline mode (save drafts locally)
- Better error messages
- Rollbar/Sentry integration
