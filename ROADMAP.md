# Crane Roadmap

## v2

### High priority
- **Email integration** — auto-detect application confirmations and interview invites, update job status automatically
- **Weekly digest** — Monday summary of what needs attention (follow-ups, deadlines, stale apps)
- **Conversion analytics** — which role titles, companies, and sources are getting responses vs silence

### Medium priority
- **Browser extension** — one-click add from any job board, not just LinkedIn
- **Resume version tracking** — track which resume was sent to which job
- **Contact tracking** — store recruiter/hiring manager names and notes per job

## Backlog
- Mobile-responsive layout
- Bulk import from CSV
- Deadline email reminders
- Dark mode
